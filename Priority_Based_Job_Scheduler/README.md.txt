# Priority-Based Job Scheduler

A Python-based job scheduling system that executes tasks according to their priority using Heap-based Priority Queue.


## Features
- Add Jobs
- Execute Highest Priority Job
- Display Jobs
- Remove Jobs
- Efficient Priority Scheduling


## Technologies Used
- Python
- heapq module
- Data Structures & Algorithms


## Data Structures Used
- Priority Queue (Heap)
- Queue
- Arrays/Lists


## Project Architecture
The system works using:
1. User Input
2. Menu Interface
3. Priority Queue (Heap)
4. Scheduling Logic
5. Job Execution


## Time Complexity

| Operation    | Complexity |
| Insert Job   | O(log n) |
| Execute Job  | O(log n) |
| Display Jobs | O(n) |
| Remove Job   | O(n) |


## How to Run

1. Open the project in Python IDLE
2. Open `scheduler.py`
3. Click `Run → Run Module`
4. Use the menu options to manage jobs


## Sample Functionalities
- Add new jobs dynamically
- Execute highest-priority tasks first
- Remove jobs from queue
- Display all jobs in priority order
